# 기여 가이드

## 작업 환경 설정

1. 저장소 클론
2. `npm install` 실행
3. `.env.example`을 복사하여 `.env` 생성
4. Firebase 설정 정보 입력
5. `npm run dev` 실행

## 코딩 규칙

- TypeScript 사용
- 함수형 컴포넌트 사용
- 의미있는 변수명 사용
- 주석은 한국어로 작성

## 커밋 메시지 규칙

- `feat: 새로운 기능 추가`
- `fix: 버그 수정`
- `docs: 문서 수정`
- `style: 코드 포맷팅`
- `refactor: 코드 리팩토링`
- `test: 테스트 추가`

## Pull Request

1. 새 브랜치 생성
2. 변경사항 커밋
3. Push
4. GitHub에서 Pull Request 생성
